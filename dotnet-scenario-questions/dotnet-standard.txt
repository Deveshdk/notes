.NET Standard is a specification (not a runtime or framework) that defines a set of common APIs that every .NET platform must implement.

In simple words:
.NET Standard is a contract that ensures code sharing across different .NET platforms.

If you target .NET Standard, your library can run on:
    .NET Framework, .NET Core, Xamarin / Mobile, Unity, Mono, UWP

ğŸ¯ Why .NET Standard Was Created?
    Before .NET Standard, platforms had incompatible APIs:
    .NET Framework had one set
    .NET Core had another
    Xamarin had another
    This made code-sharing difficult.

.NET Standard solved this by providing a unified API surface that all platforms must support.
